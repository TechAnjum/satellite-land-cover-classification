# 🛰️ ISRO Satellite Land Cover Classification

Machine learning tool to classify land cover from ISRO satellite imagery (LISS-IV / Resourcesat).

## Classes Detected
Urban | Forest | Agriculture | Water Bodies | Barren Land | Wetlands

##  Tech Stack
- **Rasterio** — Satellite image processing
- **Scikit-learn** — Random Forest classifier  
- **Folium** — Interactive maps
- **NumPy / Matplotlib** — Analysis & visualization

## 📊 Results
- Accuracy: ~95%+
- Features: NDVI, NDWI, NDBI, EVI + raw bands

## 📡 Data Source
Free satellite data from [Bhuvan Portal](https://bhuvan.nrsc.gov.in) (ISRO/NRSC)
